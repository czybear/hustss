# HUSTSS.org（website/）

华中科技大学软件学院上海校友会软件分会（筹）官网静态页面。
Apple 风格：简洁、留白、玻璃拟态、深浅色主题切换。

## 目录
本网站源码放在仓库的 `website/` 目录下：
- `website/index.html`
- `website/styles.css`
- `website/script.js`
- `website/404.html`
- `website/CNAME`
- `website/assets/`

## GitHub Pages（发布 website/）
GitHub -> Settings -> Pages
- Source: Deploy from a branch
- Branch: main
- Folder: /website

## 自定义域名
`website/CNAME` 已包含：
- `hustss.org`
